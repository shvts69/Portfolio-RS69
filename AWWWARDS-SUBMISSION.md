### Description (long, ~500–700 chars)
RS69 is the portfolio of Ross Shvts — a creative frontend developer who treats every site as a visual problem first, and code second. Instead of a grid of thumbnails, the portfolio is built as a live star-system: a volumetric WebGL Helix Nebula powers the background, three Canvas-2D galaxies orbit each case study, and a 3D model-viewer astronaut roams across the page. Hold the cursor still and it collapses into a black hole that devours the text. Selected works include a hand-pixeled band site, an MMA academy brand, and a real-production memorial platform. No framework on the shell. All interaction built from scratch.

## Where your entry will appear
- Immediately: "Nominated" list (public, gets views)
- Within 2-3 days: Awwwards staff reviews and either:
  - Rejects (rare for technically ambitious sites)
  - Assigns to daily judging pool (7+ judges score Design/Usability/Creativity/Content)
- If scored 6.5+/10: Honorable Mention
- If scored 8+/10: Site of the Day
- Technical ambition is judged separately for Developer Award
